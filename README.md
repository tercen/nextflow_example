# Nextflow Example Operator

##### Description

This operator is a wrapper around an example Nextflow pipeline (BLAST pipeline).

##### Usage

Input projection|.
---|---
`column`        | input folder path

Output relations|.
---|---
`blast_results_folder`        | Path to folder containing input files
